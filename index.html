Welcome to CI Chatsocket

Documentation By: Jack Tyler (jack@extremewebevolution.com)

Description

CodeIgniter is a proven, open source PHP web application framework that empowers the next generation of web apps and caters to everyone, from beginners to power users. "CI Chatsocket" is a chat application that uses Node JS and socket.io to communicate real time to each user. This library readily integrates with any existing Codeigniter website or application built with jQuery and Bootstrap.

Features and Function

  • Real time chat
  • Can be installed on your existing CodeIgniter Website
  • Private conversation
  • Group conversation
  • Manage group conversation users
  • Preview image and mp4 videos
  • Play youtube, vimeo directly if you paste a link
  • Manage Users who you want to have this chat enabled.
  • Convert links automatically.
  • File attachment (gif|jpg|png|zip|zipx|rar|7z|pdf|doc|docx|txt|odt|mp3|mp4)
  • Emoji (Icons)
  • Search conversations

Requirements

  • Server should have Node JS / can run Node JS
  • Php 5.4 and above (Not yet tested in 5.3)
  • MySQL server
  • Codeigniter should be built with jQuery and Bootstrap for this application to work

Installation

Follow the simple steps below

  • Copy all the files and folders inside the CIPurchasing folder and drop it to the folder where you want your website is located
  • Go to your domain installation path e.g. www.yourdomain.com/install
  • Fill up the mysql connection settings and click Install
  • Viola! You can now login using the default access

    email:admin@admin.com, password:password)

  • Kindly Setup all the settings pages first in the Admin->settings so that you can fully use the system.
